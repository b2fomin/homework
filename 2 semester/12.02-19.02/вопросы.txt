I) В каких ситуациях применяются типы std::pair и std::tuple?
Когда нужно несколько значений заранее известных типов хранить в одной
переменной, функйия должна возвращать несколько значений, возможно разных
типов и т.д.

II) Когда следует использовать контейнер std::array?
Их используют вместо стандартных фиксированных массивов. Можно использовать для
эффективной работы с заранее известным количеством данных.

III) Когда следует использовать std::vector?
Их используют вместо стандартных динамических массивов. Используется при работе
с неизвестным количеством данных.

IV) Когда следует использовать std::deque?
deque представляет собой набор небольшого размера динамических массивов, концы
которых связаны с помощью ссылок. В этой структуре данных можно эффективно 
добавлять и удалять элементы к с конца, так и с начала. Но, например, сортировка
происходит гораздо медленнее.

V) Когда используется std::list?
Данная структура данных представляет собой двухсвязный список, поэтому удаление
и добавление в любое место списка происходят довольно эффективно, но
доступ к элементам происходит гораздо медленнее, чем у обычного
массива. Также для данной структуры не нужна непрерывная область памяти для
ее хранения.

VI) Когда используется std::forward_list?
std::forward_list немного эффективнее std::list, если не требуется работы с
обоими концами спискка.

VII) Какие адаптеры контейнеров есть в стандартной библиотеке?
Адаптеры --- контейнеры, переопределенные для конкретных заданий. Это stack,
queue и priority_queue.

VIII) Когда следует использовать контейнер circular buffer из Boost?
Кольцевой буфер предоставляет возможность буферизации потоков данных. Используется
для организации различных очередей сообщений и буферов приема-передачи,
программировании микроконтроллеров.

IX) Почему контейнер circular buffer из Boost не может войти в стандарт?
Согласно стандарту, указатель на конец структуры всегда должен идти после
указателя на начало, что не всегда выполняется для данного контейнера.

X) Какие типы данных для работы с многомерными массивами вы можете назвать?
Стандартный массив C++, вложенные друг в друга контейнеры, milti_array из
библиотеки Boost.
